# Car Dealership Review Full Stack Application

A full stack, responsive web application for **Best Cars Dealership**, a national car retailer in
the United States. The application lists the company's dealerships across different states and lets
users view and submit reviews for each dealer.

## Tech stack
- **Frontend:** React, HTML, CSS, Bootstrap
- **Backend:** Django, Django REST Framework, SQLite
- **Microservices:** Node.js / Express + MongoDB (dealers & reviews), Flask sentiment analyzer
- **DevOps:** Docker, Kubernetes, IBM Cloud Code Engine, GitHub Actions (CI/CD)

## Features
- Responsive UI listing dealerships and dealer details
- User registration, login and logout
- View and post dealer reviews
- Sentiment analysis on review text
- Filter dealerships by state
